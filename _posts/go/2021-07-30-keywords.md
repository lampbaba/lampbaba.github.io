#### break
동일 함수내의 가장 안쪽에 있는 `"for"`,`"switch"` 혹은 `"select"`문의 실행을 종료한다.
```go
BreakStmt = "break" [ Label ] .
```
레이블이 있는 경우 `"for"`,`"switch"` 혹은 `"select"`문을 둘러싸고 실행이 종료되는 레이블이어야 한다.

{% highlight go linenos %}
OuterLoop:
  for i = 0; i < n; i++ {
    for j = 0; j < m; j++ {
      switch a[i][j] {
      case nil:
        state = Error
        break OuterLoop
      case item:
        state = Found
        break OuterLoop
      }
    }
  }
{% endhighlight %}
